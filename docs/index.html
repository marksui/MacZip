Features

Built for everyday archive work.

Drag and drop

Add files fast.

Drop files or folders into the app, then choose where the result should be saved.

Compress

Create clean archives.

Choose the output name, select a format, and create a new archive from one or many items.

Extract

Unpack supported files.

Extract common archive types into a folder you control, with clear success and error states.

History

See what happened.

Recent activity shows latency, throughput, CPU estimate, sizes, and compression ratio.

Formats

One focused place for common archive types.

ZIP Compress and extract
RAR Extract with unar, unrar, or 7-Zip
7Z Requires local 7z or 7zz
TAR Compress and extract
TAR.GZ Also supports TGZ
GZIP Single-file compression

Workflow

Choose, save, compress or extract.

1

Select your files.

Use the file picker or drag files and folders directly into the window.

2

Pick a destination.

Choose the folder where MarkMacZip should write the finished archive or extracted files.

3

Run the job.

Watch progress, copy the status if needed, and review the result in Recent Activity.

Privacy

Your files stay on your Mac.

No cloud upload.

No account or sign-in.

No analytics tracking.

Download

Try MarkMacZip from GitHub.

Download the macOS app from GitHub Releases. If macOS warns about an internet download, right-click the app and choose Open.
